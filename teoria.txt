1.Что такое семафоры и для чего они используются:

Семафоры это механизм синхронизации, используемый для управления доступом к общим ресурсам в многопоточных или многопроцесссных условиях, он представляет собой счётчик, который ограничивает количество потоков, которые одновременно используют ресурсб используются для предотвращения состояния гонки и организации безопасного доступа к ресурсам

2.Чем семафоры отличаются от мьютексов:

Семафоры позволяют нескольким потокам одновременно использовать ресурс а мьютекс предоставляет исключительный доступ только одному потоку
Семафор можно разблокировать любым потоком, мьютекс только его владельцем
Семафоры работают с числовым счётчиком, мьютекс бинарные либо используются или нет

3.В каких ситуациях предпочтительнее использовать семафоры:

Для ограничения количества потоков, одновременно использующих ресурс
Для синхронизации потоков в случаях производитель-потребитель
Для управления несколькими экземплярами ресурса когда мьютекс недостаточен
